

# 练习1：成绩等级判断

## 题目要求

从键盘输入一个整数成绩 `score`，范围假设为 `0~100`。
根据成绩输出对应等级：

* `90~100` 输出：`优秀`
* `80~89` 输出：`良好`
* `60~79` 输出：`及格`
* `0~59` 输出：`不及格`
* 如果输入不在 `0~100` 范围内，输出：`输入错误`

## 要求

* 必须使用 `if...else if...else`
* 自己定义变量接收输入
* 输出结果要清晰

```cpp
#include <iostream>
using namespace std;
int main()
{
    int score;
    cout << "请输入该学生的成绩" << endl;
    cin >> score;
    if (score >= 90)
    cout << "优秀" << endl;
    else if (score >= 80)
    cout << "良好" << endl;
    else if (score >= 60)
    cout << "及格" << endl;
    else 
    cout << "不及格" << endl;

    return 0;
}
```
---
评价：
**没有处理非法输入，如-1、101等**

程序改善：

# 练习4：输出 1 到 n 的和

## 题目要求

从键盘输入一个正整数 `n`，用 `for` 循环计算：

```text
1 + 2 + 3 + ... + n
```

最后输出总和。

例如输入：

```text
5
```

输出：

```text
1到5的和是：15
```

## 要求

* 必须使用 `for`
* 需要定义一个变量来保存总和，例如 `sum`

---
```cpp
#include <iostream>
using namespace std;
int main()
{
    int n;
    cin >> n;
    int sum = 0;
   
    for (int i = 0; i <= n; i++)
    {
        sum = sum + i;
    }
    cout << sum;
    return 0;
}
```
# 练习5：猜数字范围判断

## 题目要求

程序中预设一个整数，例如：

```cpp
int target = 66;
```

然后让用户输入一个数字，判断：

* 如果输入大于 `66`，输出：`猜大了`
* 如果输入小于 `66`，输出：`猜小了`
* 如果输入等于 `66`，输出：`猜对了`

## 要求

* 使用 `if...else if...else`
* 先写成“只猜一次”的版本
* 写完后可以尝试挑战升级版：用 `while` 循环让用户一直猜，直到猜对为止

---
解答：
```cpp
#include <iostream>
using namespace std;
int main()
{
    int a = 66;
    int b;
    cout << "请猜测一个数字";
    cin >> b;
    while (b != a)
    {
        if (b > a)
        {
            cout << "猜大了！";
        }
        else if (b < a)
        {
            cout << "猜小了！";
        }
       cin >> b;
    }
    if (b == a)
    cout << "恭喜你，猜中了！";
    return 0;
}
```
程序评价：
1）提示信息最好换行，如果用户多次输错，输出可能会挤在一起，不太好看。

2）每次提示完，最好再提示用户重新输入

3）最后那个 if (b == a) 其实可以省，因为能跳出 while (b != a)，就说明 b == a 已经成立了。

程序修改：
```cpp
#include <iostream>
using namespace std;

int main()
{
    int target = 66;
    int guess;

    cout << "请猜测一个数字：" << endl;
    cin >> guess;

    while (guess != target)
    {
        if (guess > target)
        {
            cout << "猜大了！" << endl;
        }
        else
        {
            cout << "猜小了！" << endl;
        }

        cout << "请继续输入：" << endl;
        cin >> guess;
    }

    cout << "恭喜你，猜中了！" << endl;

    return 0;
}
```
# 练习6：输出 1 到 100 中的偶数

## 题目要求

用循环输出 `1~100` 中所有的偶数。

输出格式不限，例如每行一个，或者空格隔开都可以。

## 要求

* 必须使用 `for`
* 你可以有两种写法：

  1. 从 1 遍历到 100，用 `if` 判断是不是偶数
  2. 直接从 2 开始，每次加 2

## 提示

判断偶数可以用：

```cpp
i % 2 == 0
```

---
代码练习：
```cpp
#include <iostream>
using namespace std;
int main()
{
    int a;
    cout << "请输入一个数，我会给出所有比它小的正偶数：";
    cin >> a;
    if (a % 2 != 0)
    {
       for ( ; a > 0; a = a - 1)
       {
           if (a % 2 == 0)
           {
             cout << a;
           }
        }
    }
    else 
    {
        a = a - 1;
        for ( ; a > 0; a = a - 1)
       {
           if (a % 2 == 0)
           {
             cout << a;
           }
          
        }

    }
    return 0;
}
```


---
一些需要注意的程序改错：

**1）`a--` 到底是什么意思？**

`a--` 叫**后置自减**，它的意思是：

> 先拿出 `a` 当前的值参与运算，
> 然后再让 `a` 自己减 1。

比如：

```cpp
int a = 5;
int b = a--;
```

执行后：

* `b = 5`
* `a = 4`

因为“先用旧值，再减一”。

---

**2）`a = a--;` 为什么有问题？**

看起来你好像想表达：

> “让 a 减 1 再赋回去”

但实际上不是这样。

`a = a--;` 可以粗糙地理解成：

1. 先取出 `a` 原来的值
2. `a` 自己减 1
3. 然后又把“原来的值”赋回给 `a`

结果就会把刚刚减掉的那一下又覆盖掉。

所以它**并不会按你想的那样稳定减 1**。
这种写法本身就很糟糕，应该避免。

只需要写：

```cpp
a--;
```

就够了。

或者：

```cpp
a = a - 1;
```

---



你的目标是：

> 输入一个数，输出所有比它小的正偶数

例如：

* 输入 9 → 输出 `8 6 4 2`
* 输入 8 → 输出 `6 4 2`

最直接的思路就是：**从 `a-1` 开始往下找，遇到偶数就输出**

代码改善：

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a;
    cout << "请输入一个数，我会给出所有比它小的正偶数：";
    cin >> a;

    for (int i = a - 1; i > 0; i--)
    {
        if (i % 2 == 0)
        {
            cout << i << " ";
        }
    }

    return 0;
}
```

---

**为什么建议不要直接改原变量 `a`**

你原来的思路一直在改 `a` 本身。
这不是绝对不行，但对初学者来说容易乱。

更推荐你这样想：

* `a` 是用户输入的原始值
* 我再用一个新变量 `i` 去循环判断

这样代码会更清楚：

* `a` 负责“记住输入值”
* `i` 负责“循环跑”

这是一个很好的编程习惯。

---

**如果想写得更巧一点，还可以直接从偶数开始跳着输出**

可以先找到“比 `a` 小的最大偶数”，然后每次减 2。

写法如下：

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a;
    cout << "请输入一个数，我会给出所有比它小的正偶数：";
    cin >> a;

    int start;
    if (a % 2 == 0)
    {
        start = a - 2;
    }
    else
    {
        start = a - 1;
    }

    for (int i = start; i > 0; i -= 2)
    {
        cout << i << " ";
    }

    return 0;
}
```

```cpp
#include <iostream>
```

```cpp
#include <iostream>

int main()
```

* 必须使用 `do...while`

---
```cpp
#include <iostream>
using namespace std;

int main()
{
    int choice;

    do
    {

                break;

                cout << "程序结束" << endl;
                break;

            default:
                cout << "无效输入" << endl;
                break;
        }

        cout << endl;
    }
    while (choice != 3);

    return 0;
}
```

# 练习10：综合题——角色行动系统

## 题目要求

设计一个小程序，角色初始能量值 `energy = 100`。
程序循环显示菜单：

```text
1.跑步（消耗10点能量）
2.休息（恢复20点能量）
3.查看能量
4.退出
```

用户输入选项后：

* 输入 `1`：能量减 10，并输出当前能量
* 输入 `2`：能量加 20，并输出当前能量
* 输入 `3`：输出当前能量
* 输入 `4`：结束程序
* 输入其他值：输出 `无效输入`

额外要求：

* 能量最低不能小于 `0`
* 能量最高不能超过 `100`

## 要求

* 使用循环（`while` 或 `do...while` 都可以）
* 使用 `switch`
* 使用 `if` 控制能量上下限

---

解答：
```cpp
#include <iostream>
using namespace std;
int main()
{   int choice;
    int energy = 100;
  
    do
    {
        cout << "1.跑步（消耗10点能量）" << endl;
        cout << "2.休息（恢复20点能量）" << endl;
        cout << "3.查看能量" << endl;
        cout << "4.退出" << endl;
        cout << "请输入选项：" << endl;

        switch (choice)
        {
            case 1:
            energy = energy - 10;
            if (energy < 0)
            {
                energy = 0;
            }
            cout << "当前能量=" << energy << endl;
            break;

            case 2:
            energy = energy + 20;
            if (energy > 100)
            {
                energy = 100;
            }
            cout << "当前能量=" << energy << endl;
            break;

            case 3:
            cout << "当前能量=" << energy << endl;
            break;

            case 4:
            cout << "结束程序" << endl;
            break;

            default:
            cout << "无效输入" << endl;
            break;
        }
    } 
    while (choice != 4);

    return 0;
}
```
#### 一、程序存在的问题一:
当输入的不是`int`类型时，`cin >> choice`会读取失败，进入“无限循环”
  
**1.错误原因** 
因为这里的 `choice` 是 `int` 类型：

```cpp
int choice;
cin >> choice;
```

而如果输入的是：

```text
/
```

`/` 不是整数，**所以`cin >> choice` 读取失败了**
一旦读取失败，`cin` 会进入**错误状态**。进入这个状态以后，如果不手动清除，它后面每一轮都会继续失败，于是程序就会表现成：

* 菜单一直打印
* `choice` 读不进去新值
* 一直走 `default`
* 看起来像“无限循环”

---

**2.错误分析** 

输入 `/` 时，程序会想：

> “我要读一个 `int`，结果你给了个字符 `/`，我没法把它转成整数。”

于是读取失败。

---
由于循环条件是：

```cpp id="03jqpd"
while (choice != 4);
```

而读取失败后：

* `choice` 通常还是之前那个值，或者保持原来的状态
* 它没有成功变成 `4`
* 所以循环条件一直成立

更关键的是：**输入流里的错误状态没有恢复**

所以后面每次执行到：

```cpp id="r7kk8t"
cin >> choice;
```
**它都还是失败，根本读不进新输入。于是就会看到程序一直转。**

---


**3.解决方法** 

需要在输入失败时做两件事：

 1. 清除错误状态
     ```cpp
    cin.clear();
     ```

2. 把这次错误输入丢掉

     ```cpp 
     cin.ignore(1000, '\n');
     ```
即：**把这一行里剩下的错误内容扔掉，别留在输入缓冲区里**

新增代码：

```cpp id="ks29kk"
if (cin.fail())
{
    cin.clear();
    cin.ignore(1000, '\n');
    cout << "无效输入" << endl;
    continue;
}
```
这整段代码的作用可以先一句话概括成：
**如果刚才输入失败了，就把错误状态清掉，把错误输入扔掉，提示“无效输入”，然后直接开始下一轮循环。**

简单解析：


```cpp 
if (cin.fail())
```

“检查一下 `cin` 有没有失败”

> 刚才输入的时候，是不是出问题了？


例如本来想输入整数：

```cpp id="2b10eu"
int choice;
cin >> choice;
```

结果输入了：

```text id="h0fphz"
/
```

那 `cin.fail()` 就会是“真”。

---


```cpp id="8m03d4"
cin.clear();
```
**清除 `cin` 的错误状态**

当你写：

```cpp id="iglr92"
cin >> choice;
```

而你输入了 `/` 的时候，`cin` 会发现：

> “我本来要读一个整数，但你给了我一个不能转成整数的东西，我失败了。”

于是 `cin` 会进入一种状态，大概像：

> “我现在出错了，我被卡住了。”

这个“卡住了”的状态，就可以理解成**错误状态**。如果不清掉这个错误状态，后面再写：

```cpp id="m2ztss"
cin >> choice;
```

它还是会继续失败。

所以这一句的作用就是：**把输入系统从“报错卡住”状态恢复过来**

---


```cpp 
cin.ignore(1000, '\n');
```

**让 `cin` 忽略掉一些输入内容**

也可以通俗理解成：

> 把一些没用的、出错的输入扔掉



`(1000, '\n')`是在告诉 `ignore`：要忽略多少、忽略到哪里为止

---

`1000`可以理解成最多忽略 1000 个字符，主要就是写个足够大的数，让这一行里的垃圾输入基本能被清掉。


`'\n'`表示的是**换行符**，也就是按下回车后产生的那个“换行”标记。

这行的意思是：**从当前输入里开始扔，最多扔掉 1000 个字符，或者扔到遇见换行为止，即把这一整行剩下的错误输入都清空**

---

#### 为什么要这样做？

假设输入的是：

```text id="l1eq9n"
/
```

然后按下回车。

实际上输入缓冲区里可能可以理解成有这些内容：

```text id="2n0j6t"
/ + 回车
```

如果你只 `clear()` 但不 `ignore()`，那个 `/` 还留在那里。
下一轮再读的时候，程序又会看到这个 `/`，于是又失败。

所以不仅要清错误状态，还要把错误内容本身也扔掉

**整体翻译：把这一行里剩下的错误输入扔掉，直到遇到回车为止**

---



```cpp id="lmjrzu"
continue;
```
**“跳过本轮后面剩下的代码，直接开始下一轮循环”**

---


现在把整段连起来：

```cpp id="ylotzt"
if (cin.fail())
{
    cin.clear();
    cin.ignore(1000, '\n');
    cout << "无效输入" << endl;
    continue;
}
```

可以翻译成：

**如果刚才输入失败了，那么：**
1. 把输入系统的错误状态清掉
2. 把这一行里错误的输入内容扔掉
3. 提示用户“无效输入”
4. 直接开始下一轮循环，让用户重新输入

---

#### 二、程序存在的问题二：
1.输入 `1abc`，虽然会报错，但能量已经先变成 `90`


```cpp id="rla1c9"
cin >> choice;
```

这里 `choice` 是 `int`，所以 `cin` 会尝试读取一个整数。输入1abc时，`cin` 会这样处理：

1. 先看开头是不是能读出整数，发现是1,可以，没问题。

2. 于是它把 `1` 读进 `choice`，这时：`choice = 1`

3. 继续往后看，发现下一个字符是`a`，这已经不是整数的一部分了，所以它**停止读取**

但是，**到这里，这次输入其实算“成功读取了一个整数 1”**

所以程序会继续往下执行：` energy -= 10;`，于是能量就从 `100` 变成了 `90`

---
但是，后面的 `abc` **没有消失**，而是留在输入缓冲区里。

所以下一轮循环再执行：

```cpp id="wkxrf0"
cin >> choice;
```

时，程序读到的就是：

```text id="wh8wnx"
abc
```

这次一上来就读不到整数，于是：

* 输入失败
* `cin.fail()` 为真
* 输出“无效输入”

---

**2.输入`1 1 1`，程序会进行3个而不是1个循环**

```cpp
cin >> choice;
```

`cin` 会把输入按“一个一个输入项”来读。



你不是“输入了一行一个东西”，而是：

**一次性给输入缓冲区塞了三个合法整数，程序每轮循环拿一个出来用**

---

**3.可以这样理解 `cin >> choice`**

它不是：

> “我读整整一行，整行必须完全正确”

而更像是：

> “我从当前输入里拿出一个能读成 `int` 的东西”

所以：

* `1abc`：先拿到 `1`
* `1 1 1`：每次拿一个 `1`
* `/`：一个整数都拿不到，立刻失败

---

**4.解决办法**
要求：

* 输入必须是“纯净的一个整数”
* `1abc` 应该整体算无效
* `1 1 1` 也应该整体算无效
* 只有输入单独一个 `1` 才算有效

代码示例：
```cpp id="yjlwmn"
cin >> choice;

if (cin.fail())
{
    cin.clear();
    cin.ignore(1000, '\n');
    cout << "无效输入" << endl;
    continue;
}

cin >> ws;   // 跳过后面的空格
if (cin.peek() != '\n')
{
    cin.ignore(1000, '\n');
    cout << "无效输入" << endl;
    continue;
}

cin.ignore(1000, '\n');
```

---

代码解析：
**1）`cin >> ws;`**
>先把后面的空格跳过去

比如输入：

```text id="7f820m"
1    
```

后面有几个空格，这句会把它们跳过。

---
**2）`cin.peek()`**
>偷看一下，下一个字符是什么，但先不真正把它拿走

---

**3）`if (cin.peek() != '\n')`**


> 如果跳过空格以后，下一个字符不是回车，说明这一行后面还有别的内容

例如：

- 输入 `1abc`
  读完 `1` 后，后面是 `a`，不是 `\n`，所以判定无效
- 输入 `1 1 1`
  读完第一个 `1` 后，跳过空格，后面还是 `1`，不是 `\n`。所以判定无效

- 输入 `1`
  读完 `1` 后，后面就是回车，所以合法

---
**5.修改好的代码**

这是**严格版**，也就是：

* `1` 合法
* `1abc` 非法
* `1 1 1` 非法
* `/` 非法

同时，将关键数字设置为常量，会更加有优势。

修改后，最终整体代码版本如下：
```cpp
#include <iostream>
using namespace std;

int main()
{
    const int RUN_COST = 10;        // 跑步消耗能量
    const int REST_GAIN = 20;       // 休息恢复能量
    const int MAX_ENERGY = 100;     // 能量上限
    const int EXIT_CHOICE = 4;      // 退出选项

    int choice;
    int energy = MAX_ENERGY;

    do
    {
        cout << "====== 菜单 ======" << endl;
        cout << "1.跑步（消耗" << RUN_COST << "点能量）" << endl;
        cout << "2.休息（恢复" << REST_GAIN << "点能量）" << endl;
        cout << "3.查看能量" << endl;
        cout << "4.退出" << endl;
        cout << "请输入选项：" << endl;

        cin >> choice;

        // 第一步：检查是否连整数都读不出来
        if (cin.fail())
        {
            cin.clear();
            cin.ignore(1000, '\n');
            cout << "无效输入：请输入整数选项" << endl;
            continue;
        }

        // 第二步：如果后面不是换行，说明这一行还有脏内容
        // 例如：1abc、1 1 1
        if (cin.peek() != '\n')
        {
            cin.ignore(1000, '\n');
            cout << "无效输入：请输入单个整数选项" << endl;
            continue;
        }

        // 第三步：把这一行剩余内容清掉（主要是回车）
        cin.ignore(1000, '\n');

        switch (choice)
        {
            case 1:
                energy -= RUN_COST;
                if (energy < 0)
                {
                    energy = 0;
                }
                cout << "当前能量 = " << energy << endl;
                break;

            case 2:
                energy += REST_GAIN;
                if (energy > MAX_ENERGY)
                {
                    energy = MAX_ENERGY;
                }
                cout << "当前能量 = " << energy << endl;
                break;

            case 3:
                cout << "当前能量 = " << energy << endl;
                break;

            case EXIT_CHOICE:
                cout << "结束程序" << endl;
                break;

            default:
                cout << "无效输入：请输入 1~4 之间的整数" << endl;
                break;
        }
    }
    while (choice != EXIT_CHOICE);

    return 0;
}
```
