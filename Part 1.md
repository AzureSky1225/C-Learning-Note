# **第一章：C++ 基础语法与数据表示**
## 一、入口函数
**程序的执行从入口函数开始**

```c
#include <iostream>
using namespace std;

int main()
{
    system ("pause");
    return 0;
}
```
以上是一个入口函数的例子，下面我们逐行解析
---
#### 1.第一行

```c
#include <iostream>
```
**“把 iostream 这个工具包拿进来。”**



1）**`#include` 是什么？**
   `#include` 的意思就是：**包含、引入**
   就像做饭前，要先把锅、铲子、调料拿出来。
   写程序也是一样的，很多功能不是凭空就能用的，要先 **“引入工具”**。


2） **`<iostream>` 是什么？**
   iostream 是 C++ 标准库里的一个头文件。
   可以把它理解成是**一个和输入、输出有关的工具包**
   
比如
- `cout`：让程序往屏幕上输出内容
- `cin`：让程序接收你输入的内容

它们都和 `iostream` 有关。
#### 2.第二行

```c
using namespace std;
```
这是一个命名空间的释明，意思大致是：
**“以后用标准库里的东西时，我就不每次都写 std:: 了。”**

**1）`std`是什么？**
  `std` 是 C++ 里的一个“命名空间”,可以把“命名空间”想象成一个分类盒子，C++ 标准库里的很多东西都放在 std 这个盒子里。

 比如

 ```c

 std::cout
 std::cin
 std::string
 ```

这里的 `std::` 就像在说：**“我用的是标准库这个盒子里的东西。”**

**2）为什么要写 `using namespace std;`**

因为如果不写这句，以后你每次都要这样写：
```c
std::cout << "Hello";
```
写了这句以后，就可以直接写：
```c
cout << "Hello";
```

#### 3.第三行
```c
int main()
```

**“这是主函数，程序从这里开始执行。”**


在 C++ 里，默认入口通常就是`main()`，也就是说，电脑运行这个程序时，会先找 main，找到了，就从这里开始往下执行。


**1）`main`是什么？**
`main` 是主函数的名字。



**2）`()` 是什么？**
括号表示这是一个函数。
数学里的函数是“给输入，出结果”，编程里的函数则更像“一个小工具”或者“一段流程”。

**3）`int` 是什么？**
`int` 表示这个函数最终会“返回一个整数”。


main 函数结束时，通常会返回一个整数给操作系统，这个整数一般用来表示程序执行结果。
最常见的是：
`return 0;` 表示程序正常结束
非 0 往往表示出现了某种问题
#### 4.第四行
```c
{
```
这个大括号表示：主函数正式开始了。
**“下面这些内容，都属于 main 这个函数内部。”**

#### 5.第七行
```c
}
```

这个大括号表示：主函数结束了。

#### 6.第五行
```c
system("pause");
```
**调用系统命令，让程序暂停。**

**1）`system` 是什么？**
system 可以理解成：**让程序去请操作系统帮忙执行一条命令**



**2）"pause" 是什么？**
"pause" 是传给系统的一条命令,意思大概就是：**暂停一下，等用户按键再继续**
所以执行后，常常会出现：**Press any key to continue...**


**3）为什么初学者喜欢写这一句？**
因为很多初学者在 Windows 下运行控制台程序时，会遇到一个问题：程序一运行完,黑框窗口立刻关闭,根本来不及看结果.所以加上：`system("pause");`就能让窗口先停住，方便看输出内容。

**4）不过，在 Linux 里，`system("pause");`通常不能用。**
#### 7.第六行
```c
return 0;
```
**主函数到这里结束，并且返回 `0`。**

**1）return 是什么？**
return 的意思是：**返回**，也就是把一个结果交回去。

**2）为什么返回 `0`？**

因为`int main()`表示这个函数结束时要返回一个整数。
这里返回 0，通常表示：**程序正常执行完毕，没有出错。**

**3）如果不写这一行可以吗？**
在很多现代 C++ 编译器里，main 函数最后即使不写 return 0;，也会默认帮你补上。但对于初学者来说，写上更清楚、更规范。
#### 8.总结
1）`main()` 是主函数,程序通常从这里开始执行。
2）`{ }` 表示函数的范围,大括号里面的内容，就是这个函数要做的事情。
3）`system("pause");` 是暂停,让黑窗口别马上关闭，等你按键。
4）`return 0;` 是正常结束，表示程序运行完了，而且没出问题。
#### 9.补充
1）严格一点说，这段代码如果要规范使用 `system()`，通常还会包含另一个头文件，比如：
```c
#include <cstdlib>
```
因为 `system()` 和这个更相关。
但很多教学环境里可能也能直接跑，所以初学资料里经常先这么写。

2）`system("pause");`这句在 Windows 里通常没问题。
但如果用的是**Linux**，这句一般会报错或者无效。

所以可以将程序改成以下的样子：
```c
#include <iostream>
using namespace std;

int main()
{
    cout << "hello world" << endl;
    cout << "你好，世界" << endl;


    return 0;
}
```
之后可以直接run，也可以打开终端，`cd "Part 1"`,然后`./main`运行，得到输出结果。

3）`endl`是换行

## 二、基本数据类型
#### **1.什么是数据类型**

在 C++ 里，数据类型就是告诉电脑：“我要存的这个东西，是什么类型的数据。”

比如：
- 10 是整数
- 3.14 是小数
- 'a' 是字符
- true 是真假值

不同类型占的内存大小不同，能存的范围也不同。


#### **2.整型**

**整型用来存储整数，不能存小数。**
|类型|占用空间|范围|
|:---|---:|:---:|
|short|2字节|-32768~32767|
|int|4字节|-2147483648~2147483647|
|long |4字节或8字节|与平台有关|
|long long  |8字节|-9223372036854775808~9223372036854775807|

**1）这些类型的区别**

- **占用内存大小不同**
  `short` 占 2 字节
`int` 占 4 字节
`long long` 占 8 字节
字节越大，通常能存的数字范围越大。



- **能表示的范围不同**
`short` 只能存比较小的整数
`int` 能存更大的整数
`long long` 能存非常大的整数

**2）为什么 36666 用 int 比 long long 更合适？**

- 因为 36666 这个数字并不大，int 完全能装下。如果一个值用 int 就够了，就没必要专门用 long long。

- 所以通常原则是：
**够用就行，不必盲目选最大类型**

**3）如何选择合适的数据类型**

- 由于实际编程里不只是看“省字节”，还要看：
1.代码可读性
2.计算效率
3.与库函数的兼容性
4.平台差异
- 比如很多时候，哪怕数字不大，程序员也会直接用 int，因为它是最常用、最自然的整数类型。
- 因此，**在满足需求的前提下，选择合适的数据类型，而不是一味追求最小字节。**

**4）整型不能存小数**

- 例如：
  ```c
  int a = 3.14;
  ```
  这里 `a` 最终不会是 `3.14`，而会变成`3`.

- 因为 int 只能保存整数部分，小数部分会被直接丢掉。
这叫做：**截断**
不是四舍五入，而是直接去掉后面的小数。

**5）示例代码**
```c
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    short b = 100;
    long long c = 1234567890123;

    cout << a << endl;
    cout << b << endl;
    cout << c << endl;

    return 0;
}
```
#### **3.浮点型（实型）**

**浮点型用来存储带小数的数据。**


| 类型       | 常见大小 |       精度      |
| :------- | ---: | :-----------: |
| `float`  |  4字节 |  约 6~7 位有效数字  |
| `double` |  8字节 | 约 15~16 位有效数字 |

---

**1）什么是有效数字**

这里的“有效数字”指的是：

> **整个数字中有意义的位数，不仅包括小数点后面的数字，也包括小数点前面的数字。**

例如：

* `3.14` 有 3 位有效数字
* `0.0314` 有 3 位有效数字
* `123.456` 有 6 位有效数字

所以更准确的说法是：

* `float`：约 6~7 位有效数字
* `double`：约 15~16 位有效数字



**2）浮点型不是完全精确的**

浮点数在计算机里有时会有精度误差。
也就是说：

> 浮点型适合表示小数，但并不保证所有小数都能完全精确表示。

例如：

```cpp
double a = 0.1;
double b = 0.2;
double c = a + b;
```

结果理论上应该是 `0.3`，但在计算机内部可能会有微小误差。



**3）float 后面常加 f**

例如：

```cpp
float a = 3.14f;
```

这里的 `f` 表示：

> 这个小数是 `float` 类型

如果不加 `f`，像 `3.14` 这种字面量默认更接近 `double`。



**4）示例代码**

```cpp
#include <iostream>
using namespace std;

int main()
{
    float a = 3.14f;
    double b = 3.1415926535;

    cout << a << endl;
    cout << b << endl;

    return 0;
}
```
---
#### **3.字符型**
**`char` 用来存储单个字符。**
|类型|占用空间|范围|
|:---|---:|:---:|
|char|1字节|存储的是字符，数字范围是-128～127|

**1）`char` 本质上存的是什么？**

- 虽然我们看到的是字符，比如：`'a'`
但在计算机内部，实际存的是这个字符对应的编码值。

- 例如在 ASCII 编码中：
`'A'` 对应 65
`'a'` 对应 97
`'0'` 对应 48

- 所以 `char` 本质上其实也是一个小整数，只不过通常拿来表示字符。

**2）字符和字符串不一样**

- 单个字符
  ```c
  char ch = 'a';
  ```
  这里只有一个字符，用单引号。

- 一串文字
  ```c
   "hello"
   ```
   这是字符串，不是 char，而是多个字符连在一起。

- 所以：
`'a'` 是字符
`"a"` 是字符串
`"hello"` 也是字符串

**3）示例代码**
```c
#include <iostream>
using namespace std;

int main()
{
    char ch = 'A';

    cout << ch << endl;
    cout << (int)ch << endl;

    return 0;
}
```
这会输出字符本身，以及它对应的数字编码。
```c
A
65
```

#### **4.bool类型**
bool 用来表示：

- 真
- 假

也就是**逻辑判断中的两种结果。**
|类型|占用空间|范围|
|:---|---:|:---:|
|bool|1字节|true或false|

**1）bool 有什么用？**
- 它特别适合表示“是否成立”。

- 例如：
今天下雨了吗？
`true` / `false`
这个人成年了吗？
`true` / `false`
条件是否满足？
`true` / `false`

**2）数字和 bool 的关系**

- 在 C++ 中，数字也可以转换成真假：
`0` 表示假
非 0 表示真

- 例如：
  ```c
  bool a = 0;   // false
  bool b = 1;   // true
  bool c = 100; // true
  ```

**3）示例代码**
```c
#include <iostream>
using namespace std;

int main()
{
    bool a = true;
    bool b = false;
    bool c = 10;

    cout << a << endl; // 输出 1
    cout << b << endl; // 输出 0
    cout << c << endl; // 输出 1

    return 0;
}
```
**4）为什么输出的是 1 和 0？**

- 因为 `cout` 输出 `bool` 时，默认常常显示：
`true` → `1`
`false` → `0`

- 如果以后想直接显示 `true` / `false`，可以用：
  ```c
  cout << boolalpha << a << endl;7
  ```
#### **5.对`cout << boolalpha << a << endl;`的解释**
**1）`cout`**

- cout 是 C++ 里用来向屏幕输出内容的东西。
可以把它理解成：**“往屏幕上打印字”**

- 比如：
   ```c
   cout << "hello";
   ```
   就会在屏幕上显示：
   ```c
   hello
   ```
**2）`<<`**
- 是 C++ 输出里常用的输出运算符。
可以理解成：**“把右边的东西送到左边的 cout 里去输出”**

- `cout << a;`
意思就是：**把 a 的值输出到屏幕**

**3）`boolalpha`**
- 它的作用是告诉 `cout`：
**以后输出 `bool` 类型时，不要显示成 `1` 和 `0`，而要显示成 `true` 和 `false`**

- 举个例子,假设：
  ```c
  bool a = true;
   bool b = false;
   ```
   默认输出
   ```c
   cout << a << endl;
    cout << b << endl;
   ```
   输出通常是：
   ```c
   1
    0
   ```
  加上 `boolalpha`
   ```c
  cout << boolalpha << a << endl;
    cout << boolalpha << b << endl;
    ```

  输出就会变成：
   ```c
  true
    false
   ```
**4）`boolalpha` 是一次性的，还是会一直生效？**

- `boolalpha` 往往不是只影响这一句,它会改变 `cout` 的输出状态。

- 比如：
  ```c
  bool a = true;
   bool b = false;
   cout << boolalpha;
   cout << a << endl;
   cout << b << endl;
   ```
  输出会是：
  ```c
  true
   false
  ```
- **一旦设置了 boolalpha，后面的 bool 输出通常也会继续按这个方式来。**
  
**5）如果想改回 1 和 0 怎么办？**

- 可以用：`noboolalpha`

- 例如：
  ```c
   cout << boolalpha << a << endl;
    cout << noboolalpha << a << endl;
  ```

  如果 `a = true`，输出会是：
  ```c
   true
    1
   ```
**5）完整小例子**
```c
#include <iostream>
using namespace std;

int main()
{
    cout << a << endl;                //1
    cout << b << endl;                //0

    cout << boolalpha;
    cout << a << endl;                //true
    cout << b << endl;                //false

    cout << noboolalpha << a << endl; //1

    return 0;
}
```
#### **6.无符号整型（unsigned）**
普通整型可以表示：

- 正数
- 负数
- 0

**但如果你确定一个值永远不会是负数，就可以考虑无符号类型。**

```c
unsigned int a = 10;
```
**1）特点**

无符号类型不能表示负数。
因为它把原本给负数用的一部分范围，全部拿去表示更大的正数了。
|类型|大小|范围|
|:---|---:|:---:|
|unsigned short|2字节|0 ~ 65535|
|unsigned int|4字节|0 ~ 4294967295|
|unsigned long|4字节或8字节|与平台有关|
|unsigned long long|8字节|0 ~ 18446744073709551615|




**2）和有符号整型对比着理解**


- `int` 范围通常是`-2147483648 ~ 2147483647`
-  `unsigned int`范围通常是`0 ~ 4294967295`
- 无符号类型把负数那一半也拿来表示正数了，所以它的最大正数更大，但最小只能到 0。
  
**3）为什么范围会变成这样？**

- 计算机底层是用二进制存数据的。
比如 1 字节 = 8 位，那么总共能表示：`2^8 = 256`种不同情况。

- 以 2 字节为例，`2 字节 = 16 位`
总情况数：`2^16 = 65536`
- 如果是 `short`
要分一半给负数，一半给正数，所以范围通常是：`-32768 ~ 32767`
- 如果是 `unsigned short`
全部都用来表示非负数，所以范围就是：`0 ~ 65535`

**4）一般公式**

- 如果一个无符号整型占 n 位，那么它的范围是：`0 ~ 2^n - 1`,其中 n 为该类型占用的二进制位数。
- 举几个例子
1字节 = 8位
`0 ~ 2^8 - 1 = 255`
2字节 = 16位
`0 ~ 2^16 - 1 = 65535`
4字节 = 32位
`0 ~ 2^32 - 1 = 4294967295`
8字节 = 64位
`0 ~ 2^64 - 1 = 18446744073709551615`

**5）无符号类型怎么写？**

- 在 C++ 里常见写法有：
   ```c
  unsigned short a;
    unsigned int b;
    unsigned long c;
    unsigned long long d;
  ```
- 例如：
  ```c
  unsigned int age = 20;
  ```
  意思是：
定义一个无符号整型变量 `age`，它只能存 `0` 和正整数。

**6）`unsigned` 单独写时通常默认是 `unsigned int`**

- 比如：`unsigned a = 10;`
通常等价于：`unsigned int a = 10;`


#### **7.sizeof 关键字


`sizeof` 用来查看某种数据类型或某个变量在内存中占多少字节。

---

**1）基本用法**

```cpp
sizeof(数据类型)
sizeof(变量名)
```

例如：

```cpp
cout << sizeof(int) << endl;
cout << sizeof(double) << endl;
```


**2）示例代码**

```cpp
#include <iostream>
using namespace std;

int main()
{
    cout << "short 占 " << sizeof(short) << " 字节" << endl;
    cout << "int 占 " << sizeof(int) << " 字节" << endl;
    cout << "long 占 " << sizeof(long) << " 字节" << endl;
    cout << "long long 占 " << sizeof(long long) << " 字节" << endl;

    return 0;
}
```


**3）说明**

不同编译器、不同系统平台下，某些类型的大小可能不同。
例如 `long` 在不同平台下可能是 4 字节，也可能是 8 字节。
所以实际大小最好通过 `sizeof` 来查看。

#### **8.转义字符**

**2）最常用的转义字符**

* `\n`

---

**3）示例代码**

```cpp
#include <iostream>

cout << "\n" << endl;
```

表示先输出一个换行符。

#### **9.字符串型


字符串用于表示**一串字符**。

例如：

* `"hello"`
* `"张三"`
* `"C++"`

---

**1）常见的两种字符串风格**

C++ 中常见有两种字符串写法：

1. **C 风格字符串**
2. **C++ 风格字符串**


**（1）C 风格字符串**

定义格式：

```cpp
char 变量名[] = "字符串值";
```

例如：

```cpp
char str1[] = "hello world";
```

## 三、变量和常量

#### **1.变量**

在 C++ 里，**变量**就是程序中一个**可以被改变的量**。

可以把变量理解成：

* 一个有名字的“盒子”
* 这个盒子里可以存放数据
* 盒子里的值以后还可以修改

比如：

* `int a = 10;`
* `a` 是变量名
* `10` 是当前存进去的值

变量的值在程序运行过程中可以不断变化。

---

**1）变量的定义**

变量的一般定义格式是：

```cpp
数据类型 标识符;
 数据类型 标识符 = 值;
```

例如：

```cpp
int a;
 int b = 10;
 float c = 3.14f;
 char d = 'A';
 bool e = true;
```

* `int a;` 表示定义了一个整型变量 `a`
* `int b = 10;` 表示定义变量 `b` 的同时，给它赋初值 `10`

---

**2）变量赋值**

变量定义之后，可以给它赋值，也可以多次赋值。

```cpp
int a;  // 这时候a已经被定义了，但值通常是不确定的
 a = 10;  // 赋值
 cout << a << endl; // 输出、换行
 a = 20;  // 再次赋值，会覆盖之前的值
 cout << a << endl; // 输出、换行
```

输出：

```text
10
 20
```

* 第一次赋值后，`a = 10`
* 第二次赋值后，原来的 `10` 被覆盖，变成了 `20`

---

**3）未初始化变量**

* 例如：

```cpp
int a;
 cout << a << endl;
```

* 这种写法虽然有时能运行，但 `a` 的值通常是**不确定的**。

* 也就是说，它不是“空的”，而是内存中原本残留的某个值。

* 所以初学时最好养成习惯：
  **定义变量时尽量初始化**

例如：

```cpp
int a = 0;
```

这样更安全，也更清晰。

---

**4）常见变量示例**

```cpp
int main()
{
    int a = 10;
    float b = 3.14f;
    double c = 1.2345;
    char d = 'f';
    bool e = true;

    cout << a << "\t" << b << "\t" << c << "\t" << d << "\t" << e << endl;
    return 0;
}
```

输出：

```text
10      3.14    1.2345  f       1
```

* `\t` 表示**制表符**，可以理解为“空出一段距离，便于对齐”
* `bool` 默认输出通常是：`true` 输出为 `1`，`false` 输出为 `0`

---

**5）变量可以参与运算**

**（1）同类型变量相加**

```cpp
int a = 10;
 int b = 20;
 int c = a + b;
 cout << c << endl;
```

输出：

```text
30
```

---

**（2）不同类型变量参与运算**

```cpp
int a = 10;
 float b = 20.22;
 int c = a + b;
 cout << c << endl;
```

输出：

```text
30
```

* `a + b` 的结果本来是 `30.22`，但是 `c` 是 `int` 类型。 `int` 不能保存小数，所以小数部分会被直接舍弃

* 这叫：**截断**
  不是四舍五入，而是直接去掉小数部分。

---

**（3）如果结果用浮点型接收**

```cpp
int a = 10;
 float b = 20.22;
 float c = a + b;
 cout << c << endl;
```

输出：

```text
30.22
```

* 因为 `float` 可以保存小数部分，所以结果不会丢失。

---

**6）初始化与赋值**

**（1）初始化**

* 指的是：**变量在定义的同时给它一个初始值**

例如：

```cpp
int a = 10;
```

这里既完成了定义，也完成了初始化。

---

**（2）赋值**

* 指的是：变量已经存在，再给它一个新值

例如：

```cpp
int a;
 a = 10;
```

这里：

* `int a;` 是定义
* `a = 10;` 是赋值

#### **2.常量**

在 C++ 里，**常量**就是程序中**不可以被改变的量**。

可以把常量理解成：

* 一个有名字的盒子
* 但这个盒子一旦放进去某个值，就不能再改

---
**1）const 常量的定义**

定义格式：

```cpp
const 数据类型 标识符 = 值;
```

例如：

```cpp
const int a = 10;
 cout << a << endl;
```

---

**2）const 常量必须初始化**

* 下面这种写法是错误的：

```cpp
const int a;
```

* 因为常量一旦定义，就必须立刻有确定的值。

* 正确写法：

```cpp
const int a = 10;
```

---

**3）常量不能被修改**

例如：

```cpp
const float pi = 3.14f;
```

如果后面再写：

```cpp
pi = 3.1415926f;
```

会报错，因为这个量已经被定义为常量，不能再改。

如果想修改它，只能回到定义处改：

```cpp
const float pi = 3.1415926f;
```

---

**4）为什么要使用常量**

* 如果某个值在程序中**固定不变**，就适合定义为常量
* 这样可以防止误修改
* 也能让代码更容易理解

例如：

```cpp
const int DAYS_PER_WEEK = 7;
```

比直接在代码里到处写 `7` 更清楚。

---

**常量的好处：**

1. **防止误修改**
2. **代码更清晰**
3. **后期修改方便**

如果程序中很多地方都要用圆周率，只需要改一处即可。

---

**5）变量和常量的区别**

* **变量**：值可以改
* **常量**：值不能改

例如：

```cpp
int a = 10;
a = 20;   // 正确，变量可以修改
```

```cpp
const int b = 10;
b = 20;   // 错误，常量不能修改
```

---

**6）字面常量**

除了 `const` 定义出来的常量，还有一种常见常量叫**字面常量**。

例如：

```cpp

3.14
'a'
true
"hello"
```

这些直接写出来的固定值，也都是常量。

例如：

```cpp

```

这里的 `10` 就是一个字面常量。

#### **3.关于 cin**

`cin` 用来**从键盘输入数据**。

基本写法：

```cpp
int a;
 cin >> a;
```

表示：

* 从键盘输入一个整数
* 把输入的值存到变量 `a` 中

---

**1）一次输入多个变量**

```cpp
int a;
 int b;
 cin >> a >> b;
```

表示：

* 从键盘按顺序输入两个整数
* 第一个赋值给 `a`
* 第二个赋值给 `b`

如果输入：

```text
12 34
```

那么：

* `a = 12`
* `b = 34`

如果再输出：

```cpp
cout << a << "\t" << b << endl;
```

结果为：

```text
12      34
```

---

**2）输入顺序要对应**

例如：

```cpp
int age;
int score;
cin >> age >> score;
```

如果输入：

```text
18 95
```

那么：

* `age = 18`
* `score = 95`

输入顺序必须和变量顺序对应。

---

**3）空格和回车都可以分隔输入**

例如下面两种输入效果通常一样：

```text
12 34
```

或者：

```text
12
34
```

## 四、标识符和关键字

#### **1.标识符**

在 C++ 里，**标识符**就是程序员自己取的名字，用来表示：

正确：

错误：

这种命名法现在没有以前那么常用了，了解即可。

**4）取余（取模）运算符 `%`**

```cpp 
10 % 3
```

**（1）取模的注意事项**

```cpp id="pss7zb"
int a = 10;
```

```cpp id="9lq6uz"
double d1 = 3.14;
```

**5）递增与递减**

递增和递减用于让变量快速加 1 或减 1。

常见运算符有：

* `++` ：递增
* `--` ：递减

例如：

```cpp
a++;
```

相当于：

```cpp id="xm4y90"
a = a + 1;
```

例如：

```cpp id="vwqc7o"
a--;
```

相当于：

```cpp id="s2kn9a"
a = a - 1;
```

---

**（1）后置递增**

```cpp 
a++;
```

表示先使用 `a` 当前的值，再让 `a` 自增 1。

示例：

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    a++;
    cout << a << endl;

    return 0;
}
```

输出：

```text
11
```

---

**（2）前置递增**

```cpp 
++a;
```

表示先让 `a` 加 1，再使用新值。

示例：

```cpp 
#include <iostream>
using namespace std;

int main()
{
    int b = 10;
    ++b;
    cout << b << endl;

    return 0;
}
```

输出：

```text
11
```

---

**（3）前置和后置的区别**

如果只是单独写：

```cpp 
a++;
++a;
```

那它们的效果看起来都像是“加 1”。

但如果把它们放到表达式里，它们就有区别了。

```cpp 
#include <iostream>
using namespace std;

int main()
{
    // 后置递增
    int a = 10;
    int b = a++ * 10;
    cout << b << endl;

    // 前置递增
    int a2 = 10;
    int b2 = ++a2 * 10;
    cout << b2 << endl;

    return 0;
}
```

输出：

```text
100
110
```

---

**（4）为什么结果不一样**

**后置递增：先用，后加**

```cpp 
int a = 10;
int b = a++ * 10;
```

执行过程可以理解为：

1. 先把 `a` 当前的值 `10` 参与计算
2. `10 * 10 = 100`
3. 然后 `a` 再自己加 1，变成 `11`

所以：

* `b = 100`
* `a = 11`

---

**前置递增：先加，后用**

```cpp 
int a2 = 10;
int b2 = ++a2 * 10;
```

执行过程可以理解为：

1. 先把 `a2` 加 1，变成 `11`
2. 再让 `11` 参与计算
3. `11 * 10 = 110`

所以：

* `b2 = 110`
* `a2 = 11`

---

**（5）递减也是同样的道理**

* `--a` ：前置递减，先减再用
* `a--` ：后置递减，先用再减

```cpp 
int a = 2;
int b = --a;
```

结果：

* `a = 1`
* `b = 1`



```cpp 
int a = 2;
int b = a--;
```

结果：

* `a = 1`
* `b = 2`

---

**（6）示例代码

```cpp id="2ftr3r"
#include <iostream>
using namespace std;

int main()
{
    // 后置递增
    int a = 10;
    a++;    // 等价于 a = a + 1
    cout << a << endl;   // 11

    // 前置递增
    int b = 10;
    ++b;
    cout << b << endl;   // 11

    // 区别
    // 前置递增先对变量进行 ++，再计算表达式
    int a2 = 10;
    int b2 = ++a2 * 10;
    cout << b2 << endl;

    // 后置递增先计算表达式，后对变量进行 ++
    int a3 = 10;
    int b3 = a3++ * 10;
    cout << b3 << endl;

    return 0;
}
```

**2.关系运算符**

**1）什么是关系运算符**

关系运算符也叫**比较运算符**，用来比较两个值之间的关系。

比较的结果只有两种：

* 真（`true`）
* 假（`false`）

---

**1）常见关系运算符**

| 运算符  |  含义  |    运算符  |  含义  |
| :--- | :--: |  :--- | :--: |
| `>`  |  大于   |  `!=` |  不等于 |
| `<`  |  小于  |  `>=` | 大于等于 |
| `==` |  等于  |  `<=` | 小于等于 |


---

**2）关系运算符的判断规则**

* `a > b` ：如果左边大于右边，则为真
* `a < b` ：如果左边小于右边，则为真
* `a == b` ：如果左右两边相等，则为真
* `a != b` ：如果左右两边不相等，则为真
* `a >= b` ：如果左边大于或等于右边，则为真
* `a <= b` ：如果左边小于或等于右边，则为真

---

**3）关系运算的结果是 bool 类型**

例如：

```cpp
bool isRight = 1 == 1;
cout << isRight << endl;
```

输出：

```text
1
```

这里要注意：

* `1 == 1` 是真的
* 所以 `isRight` 的值是 `true`
* `cout` 输出 `bool` 时，默认显示为 `1` 和 `0`

  * `true` 显示为 `1`
  * `false` 显示为 `0`

---

**4）示例代码**

```cpp
#include <iostream>
using namespace std;

int main()
{
    cout << (5 > 3) << endl;
    cout << (5 < 3) << endl;
    cout << (5 == 5) << endl;
    cout << (5 != 5) << endl;
    cout << (5 >= 5) << endl;
    cout << (3 <= 5) << endl;

    return 0;
}
```

输出：

```text
1
0
1
0
1
1
```

---

**5）注意：`=` 和 `==` 不一样**

这个是初学者最容易写错的地方。

* `=` 表示**赋值**
* `==` 表示**判断是否相等**

例如：

```cpp
int a = 10;
```

这是把 `10` 赋值给 `a`

而：

```cpp
a == 10
```

这是在判断 `a` 是否等于 `10`

#### 2.逻辑运算符


逻辑运算符用来连接或处理多个条件。

常见有：

* `&&` ：逻辑与
* `||` ：逻辑或
* `!` ：逻辑非

逻辑运算的结果也是：

* `true`
* `false`

---

**1）逻辑与 `&&`**

只有两个条件**都为真**，结果才为真。

```cpp
bool is = 1 > 1 && 2 < 3;
cout << is << endl;
```

输出：

```text
0
```

解释：

* `1 > 1` 是假
* `2 < 3` 是真
* 假 && 真 → 假

---

```cpp
bool is = 1 == 1 && 2 < 3;
cout << is << endl;
```

输出：

```text
1
```

解释：

* `1 == 1` 是真
* `2 < 3` 是真
* 真 && 真 → 真

---

**2）逻辑或 `||`**

只要两个条件中**至少有一个为真**，结果就为真。

```cpp
bool is = 1 > 1 || 2 < 3;
cout << is << endl;
```

输出：

```text
1
```

解释：

* `1 > 1` 是假
* `2 < 3` 是真
* 假 || 真 → 真

---

**3）逻辑非 `!`**

逻辑非表示“取反”。

* 原来是真，就变成假
* 原来是假，就变成真

```cpp
bool is = !1;
cout << is << endl;
```

输出：

```text
0
```

因为：

* `1` 在逻辑判断中表示真
* `!1` 就是假

---

```cpp
bool is = !0;
cout << is << endl;
```

输出：

```text
1
```

因为：

* `0` 在逻辑判断中表示假
* `!0` 就是真

---

**4）逻辑运算中的真假规则**

在 C++ 中：

* `0` 表示假
* **非 0** 表示真

所以：

* `!0` → `true`
* `!1` → `false`
* `!100` → `false`

---

**5）逻辑运算示例**

```cpp
#include <iostream>
using namespace std;

int main()
{
    cout << (1 == 1 && 2 < 3) << endl;
    cout << (1 > 1 || 2 < 3) << endl;
    cout << (!0) << endl;
    cout << (!1) << endl;

    return 0;
}
```

输出：

```text
1
1
1
0
```

#### 3.位运算符
**1）什么是位运算符**

位运算符是直接对整数的**二进制位**进行操作的运算符。

常见有：

* `&` ：按位与
* `|` ：按位或
* `^` ：按位异或
* `~` ：按位取反
* `<<` ：左移
* `>>` ：右移

---

**2）位运算是针对二进制进行的**

例如十进制的 `10`，写成二进制是：

```text
1010
```

所以位运算本质上不是直接看十进制，而是看二进制的每一位。

---
**3）按位与 `&`**

规则：

* 两个位都为 `1`，结果才是 `1`
* 其余情况都是 `0`

例如：

```text
a = 10101011
b = 11000100
```

则：

```text
a & b = 10000000
```

---

**4）按位或 `|`**

规则：

* 两个位都为 `0`，结果才是 `0`
* 其余情况都是 `1`

例如：

```text
a = 10101011
b = 11000100
```

则：

```text
a | b = 11101111
```

---

**5）按位异或 `^`**

规则：

* 两个位**不同**，结果为 `1`
* 两个位**相同**，结果为 `0`

例如：

```text
a = 10101011
b = 11000100
```

则：

```text
a ^ b = 01101111
```

---

**6）按位取反 `~`**

规则：

* `1` 变成 `0`
* `0` 变成 `1`

例如：

```text
a = 10101011
~a = 01010100
```

不过要注意：

* 在程序里，整数通常是按固定字节数存储的
* 所以实际结果会和“只看这 8 位”略有差别
* 初学阶段先理解规则即可

---

**7）左移 `<<`**

左移表示：所有二进制位整体向左移动若干位，右侧补 `0`

例如：

```text
10101011 << 2
```

结果可以理解为：

```text
1010101100
```

在固定字节数下，高位超出的部分会被丢掉。

---
**8）右移 `>>`**

右移表示：所有二进制位整体向右移动若干位。

例如：

```text
10101011 >> 2
```

结果可以理解为：

```text
00101010
```

* 被移出去的位会被丢掉
* 左边补什么，和类型以及实现有关
* 对初学者来说，先记住“右移相当于整体往右挪”就够了

---

**9）位运算示例**

```cpp
#include <iostream>
using namespace std;

int main()
{
    int value1 = 10;           // 二进制：1010
    int value2 = value1 >> 2;  // 二进制：0010
    cout << value2 << endl;    // 输出 2

    int value3 = value1 ^ 2;   // 1010 ^ 0010 = 1000
    cout << value3 << endl;    // 输出 8

    return 0;
}
```

输出：

```text
2
8
```

---
**10）关于位运算的说明**

* 位运算在初学阶段使用相对较少
* 但它在底层开发、性能优化、权限控制、图像处理等场景中很常见
* 现阶段先掌握基本规则即可

#### 5.条件运算符


条件运算符也叫**三目运算符**，写法是：

```cpp
条件 ? 值1 : 值2
```

它的意思是：

* 如果条件成立，就取 `值1`
* 如果条件不成立，就取 `值2`

---

**1）基本示例**

```cpp
int a = 10;
int b = 20;
int max = (a > b) ? a : b;
cout << max << endl;
```

输出：

```text
20
```

解释：

* 判断 `a > b` 是否成立
* 如果成立，`max = a`
* 如果不成立，`max = b`

---

**2）条件运算符的作用**

条件运算符适合写一些简单的“二选一”判断。

例如：

```cpp
int score = 85;
string result = (score >= 60) ? "及格" : "不及格";
```

#### 6.运算符优先级

**1）什么是运算符优先级**

当一个表达式中出现多个运算符时，C++ 会按照一定顺序来计算。

例如：

```cpp
cout << 2 + 3 * 4 << endl;
```

输出：

```text
14
```

因为乘法 `*` 的优先级高于加法 `+`，所以先算：

```text
3 * 4 = 12
2 + 12 = 14
```

---

**2）可以用括号改变优先级**

例如：

```cpp
cout << (2 + 3) * 4 << endl;
```

输出：

```text
20
```

因为括号里的内容会优先计算。

---

**3）初学阶段常见优先级规律**

可以先记住下面这些常见规律：

1. 括号 `()` 优先级最高
2. 单目运算（如 `!`、`~`）优先级较高
3. 乘除取余 `* / %` 高于 加减 `+ -`
4. 比较运算通常低于算术运算
5. 逻辑与 `&&` 高于 逻辑或 `||`
6. 赋值运算优先级较低

---

**4）初学建议**

* 表达式复杂时，不要只依赖优先级去猜
* **多用括号让表达更清楚**

例如：

```cpp
int result = (a + b) * c;
```

通常比不写括号更清晰。

---

#### 7.总结

* **算术运算符**：用于加减乘除和取余
* **关系运算符**：用于比较大小，结果是真或假
* **逻辑运算符**：用于连接多个条件
* **位运算符**：直接对二进制位进行操作
* **赋值运算符**：把值赋给变量，复合赋值可简化写法
* **条件运算符**：适合简单的二选一判断
* **运算符有优先级**：表达式复杂时，可以用括号改变运算顺序
